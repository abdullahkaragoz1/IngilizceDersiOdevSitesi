1. Software which is easy to use is ..user-friendly..

A.user-easy    B.user-friendly    C.usable   

2. Software which is obvious to use is..comprehensible..

A.intuitive    B.guessable    C.comprehensible   

3. Software which is not obvious to use is ..non-intuitive..

A.counter-intuitive    B.unintuitive    C.non-intuitive   

4. Software for use by children and schools is ..educational..

A.learning    B.teaching    C.educational   

5. Software for use by businesses is ..commercial..

A.commercial    B.businesslike    C.busy   

6. Software made specially for one company is ..unique..

A.one-off    B.unique    C.tailor-made   

7. Software for use at home is ..for home use..

A.for home use    B.for house use    C.for household use   

8. Software which has been illegally copied is ..pirated..

A.unreal    B.pirated    C.fake   

9. Software which has been bought from the company that produced it is ..licenced..

A.real    B.justified    C.licenced
