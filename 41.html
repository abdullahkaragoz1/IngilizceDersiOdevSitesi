30. Match each term with the correct definition



    1. trial version             2. shareware                   3. freeware  
    4. home-use version   5. professional version  



1. A simplified version which is cheaper to buy. = home-use version

2. Software which is in the public domain. Anybody can use it without paying = freeware

3. The full version with all the features. = professional version

4. You can try it for a while for free. Then if you want to keep using it, you are
expected to pay a small fee to the writer. = trial version

5. You can use it for free for a while (often a month). When the trial period finishes,
you have to pay, or the program will de-activate. = shareware




30. Her terimi doğru tanımla eşleştirin.



    1. Deneme sürümü          2. kısıtlı yazılım                 3. ücretsiz yazılım  
    4. ev-kullanımı sürümü   5. profesyonel sürüm  



1. Satın almak için daha ucuz olan basitleştirilmiş bir versiyon. = ev-kullanım sürümü

2. Kamu alanında bulunan yazılım. Kimse ödemeden kullanabilir = ücretsiz

3. Tüm özellikleri ile tam sürüm. = profesyonel sürüm

4. Bir süreliğine bedava deneyebilirsiniz. O halde kullanmaya devam etmek istiyorsanız
yazarına küçük bir ücret ödemesi bekleniyor. = deneme sürümü

5. Bir süreliğine ücretsiz kullanabilirsiniz (genellikle bir ay). Deneme süresi bittiğinde,
  ödemeniz gerekiyor veya program devre dışı bırakılacak. = kısıtlı yazılım
