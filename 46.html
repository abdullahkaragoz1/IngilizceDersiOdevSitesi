46. Applications



    application  - firewall  - program  - update  
    pirated  - licence  - security code  
    driver  - photo album  - software  



1. programs which tell the computer what to do = software

2. a piece of software which makes a computer do a task
(for example, edit an image) = application

3. any set of instructions for a computer = application

4. software which operates a peripheral, such as a scanner or printer = driver

5. application which stores and displays digital photos = photo album

6. you enter a security code to prove that
you have a __security code__ to use the software

7. software which prevents unauthorised access
to your computer over the internet = firewall

8. a series of letters and numbers which you have to enter before installing some programs = licence

9. download new features for an application = update

10. an unauthorised copy of a program = pirted




46. Uygulamalar



    uygulama  - güvenlik duvarı  - program  - güncelleme  
    korsan  - lisans  - güvenlik kodu  
    sürücü  - fotoğraf albümü  - yazılım  



1. bilgisayara ne yapacağını söyleyen programlar = __yazılım__

2. Bir bilgisayarı bir görev yapan bir yazılım parçası
(örneğin, bir resmi düzenleyin) = __program__

3. bilgisayar için herhangi bir talimat kümesi = uygulama

4. tarayıcı veya yazıcı gibi bir çevre birimi çalıştıran yazılım = sürücü

5. dijital fotoğrafları saklayan ve görüntüleyen uygulama = fotoğraf albümü

6. Bunu kanıtlamak için bir güvenlik kodu girersiniz yazılımı kullanmak için bir __güvenlik kodu__ var

7. yetkisiz erişimi engelleyen yazılım
internete bağlı bilgisayarına = __güvenlik duvarı__

8. Girmeniz gereken bir dizi harf ve sayı   bazı programları yüklemeden önce = lisans

9. Bir uygulama için yeni özellikler indir = güncelleme

10. bir programın izinsiz bir kopyası = korsan
